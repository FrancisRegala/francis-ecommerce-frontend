## â–¶ï¸ Running the Project Locally

This is a **static frontend project**, so no build step is required.

### Option 1: Using a Local Web Server (Recommended)

Serve the project root directory using your preferred local server.

Once running, open your browser and navigate to:


You may also open the compiled file directly:

```text

```

However, using a local server is recommended to avoid any browser security restrictions.
